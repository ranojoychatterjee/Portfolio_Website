PH.D Candidate in Health Data Science, Milken Institute School of Public Health
The Rahnavard Lab

Concentration: Machine Learning, Single-cell RNA seq, Graph Neural Network.

    Thesis: Characterizing the cell and gene dynamics using a single cell experiment
I developed cellSight, an R-based pipeline designed to automate multiple stages of single-cell RNA sequencing (scRNA-seq) analysis. This tool handles gene expression acquisition, quality control, data integration, differential expression analysis (DEA), and intercellular communication assessments. The pipeline simplifies workflows for researchers by streamlining tasks such as filtering poor-quality cells, normalizing expression levels, and identifying genes differentially expressed across conditions. It also evaluates cell-to-cell communication (CCC), offering insights into how cells interact within tissues. I also created a graphical neural network (GNN) model that predicts ligand-receptor interactions based on spatial transcriptomics and gene expression data, allowing researchers to map communication networks between cells. In addition, I built a deep learning model that automatically classifies cell types in scRNA-seq datasets by analyzing gene expression patterns. This model provides a highly accurate, efficient method for identifying specific cell populations, improving the speed and reliability of single-cell analysis. These innovations significantly reduce the complexity of scRNA-seq analysis, enhancing accuracy and making high-level bioinformatics tools more accessible.
